Animation-timing-function

Elle spécifie comment une animation progresse au fil du temps. Par exemple, l’animation peut accélérer, ralentir, ou même rebondir à certains moments j'ai rajouter a l'exemple animation-iteration-count: infinite; pour qu'on puisse bien voir l'animation.

Exemple d'utilisation

Code

@keyframes rotateAndScale {
    0% { transform: rotate(0deg) scale(1); }
     50% { transform: rotate(180deg) scale(0.5); }
     100% { transform: rotate(360deg) scale(1); }
}
                            
.animation_timing1 {
    width: 100px;
    height: 100px;
    background: linear-gradient(to right, orange, orangered);
    animation-name: rotateAndScale;
    animation-duration: 4s;
    animation-timing-function: ease-in-out;
    animation-iteration-count: infinite;
}
                      
@keyframes bounce {
    0% { transform: scale(1,1) translateY(0); }
    50% { transform: scale(1.1, 0.9) translateY(0); }
    70% { transform: scale(0.9, 1.1) translateY(-100px); }
    100% { transform: scale(1,1) translateY(0); }
}
                         
.animation_timing2 {
    width: 100px;
    height: 100px;
    background: linear-gradient(to right, #561CD4, #7D3C98);
    animation-name: bounce;
    animation-duration: 2s;
    animation-timing-function: cubic-bezier(0.680, -0.550, 0.265, 1.550); 
    animation-iteration-count: infinite;
}

Box-Shadow

La propriété box-Shadow permet d'ajouter une ombre à un élément. Elle est utilisée pour créer des effets de profondeur et de dimensionnalité, en simulant une source de lumière qui projette une ombre derrière l'élément.

Code

.boxshadow{
border-radius: 10px;
height: 50px;
width: 50px;
background-color:#561CD4;
box-shadow: 10px 10px 5px grey;
}

Columns count

La propriété column-count définit le nombre de colonnes dans lesquelles le contenu d'un élément doit être divisé. En utilisant cette propriété, le texte et les autres éléments enfants sont automatiquement répartis sur le nombre de colonnes spécifié.

Exemple d'utilisation

Code

.columns{
    column-count: 3;
    column-gap: 20px;
}

Flex

La propriété display: flex permet de créer un conteneur flexible pour organiser facilement les éléments enfants. En utilisant display: flex, tu peux aligner et distribuer les éléments horizontalement ou verticalement avec beaucoup de flexibilité. C'est super utile pour créer des mises en page réactives et adaptatives.

Exemple d'utilisation

Code

.flex {
    display: flex;
    justify-content: center; 
    align-items: center; 
    height: 275px;
    width: 275px;
    background-color: black;
}
                            
.flex :first-child {
    padding: 4REM;
    background-color: white;
    justify-content: center;
    align-items: center;
}

Grid

La propriété grid est une propriété abrégée en CSS qui permet de configurer un conteneur de grille et ses éléments enfants en une seule déclaration. Elle offre un moyen puissant et flexible de créer des mises en page complexes à deux dimensions. Avec la propriété grid, on peut définir à la fois les lignes et les colonnes de la grille, ainsi que les espaces entre elles.

Exemple d'utilisation

Code

.grid{
    display: grid;
    padding: 1rem;
    gap: 1rem;
    grid-template-areas:
    'grid1 grid1 grid1'
    'grid2 grid3 grid4'
    'grid5 grid6 grid6';
    height: 500px; 
    width: 350px; 
    border: 1px solid #561CD4;
}
                            
.grid :first-child{
    background-color: #fbc2eb;
    grid-area: grid1;
}
.grid :nth-child(2){
    background-color: #d8c2fb;
    grid-area: grid2;
}
.grid :nth-child(3){
    background-color: #fbc2c2;
    grid-area: grid3;
}
.grid :nth-child(4){
    background-color: #dafbc2;
    grid-area: grid4;
}
.grid :nth-child(5){
    background-color: #95a1e9;
    grid-area: grid5;
}
.grid :last-child{
    background-color: #82d3a0;
    grid-area: grid6;
}

Mix-blend-mode

La propriété CSS mix-blend-mode détermine comment le contenu d’un élément se mélange avec le contenu de son élément parent et de son arrière-plan. Elle permet de créer des effets visuels intéressants en spécifiant comment les couleurs des éléments qui se chevauchent interagissent. Les valeurs possibles pour cette propriété incluent : normal, multiply, screen, overlay, darken, lighten, color-dodge, color-burn, hard-light, soft-light, difference, exclusion, hue, saturation, color, et luminosity. Il est important de noter que mix-blend-mode n’est pas encore pris en charge par tous les navigateurs.

Exemple d'utilisation

Code

.borderimage {
    border: 20px solid transparent;
    padding: 15px;
    border-image: url(/assets/img/bg_borderimage.jpg) 30 stretch;
}
Résultat

height: 0;

Transitions

Les transitions permettent de contrôler la vitesse de changement d’une propriété CSS. Par exemple, transition: width 2s; change la largeur de l’élément sur une période de 2 secondes lors du survol. Nous pouvons spécifier sur qu'elle éléments comme dans cette exemple le width ou l'appliquer sur tous avec all

Exemple d'utilisation

Code

.transition1 {
    width: 100px;
    height: 100px;
    background-color: orangered;
    transition: width 2s;
}
.transition1:hover{
    width: 150px;
}
.transition2 {
    width: 100px;
    height: 100px;
    background-color:#561CD4;
    transition: transform 2s;
}
.transition2:hover {
    transform: rotate(180deg);
background-image: linear-gradient(to top, #a18cd1 0%, #fbc2eb 100%);
}

Text décoration

La propriété text-decoration est utilisée pour ajouter des décorations au texte, telles que des lignes sous, au-dessus ou à travers le texte. Elle est une propriété abrégée qui permet de configurer simultanément text-decoration-line, text-decoration-color, et text-decoration-style.

Exemple d'utilisation

Code

.text-decoration :first-child {
    text-decoration: underline dotted red;
    font-size: 1.4rem;
}
.text-decoration :last-child{
    text-decoration: line-through wavy blue;
    font-size: 1.4rem;
}
Résultat
