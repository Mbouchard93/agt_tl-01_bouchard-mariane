Border Radius

Cette propriété permet d’arrondir les coins d’un élément. Elle accepte jusqu’à quatre valeurs, qui correspondent aux coins dans l’ordre : haut gauche, haut droit, bas droit et bas gauche, un seul paramètre arrondiras tout les coin égaux.

Exemple d'utilisation

Code

.borderradius{
border-radius: 10px 20px 30px 40px;
height: 50px;
width: 50px;
background-color:#561CD4;
};
Résultat

MDN - border radius

Box-Shadow

La propriété box-Shadow permet d'ajouter une ombre à un élément. Elle est utilisée pour créer des effets de profondeur et de dimensionnalité, en simulant une source de lumière qui projette une ombre derrière l'élément.

Exemple d'utilisation

Code

.boxshadow{
border-radius: 10px;
height: 50px;
width: 50px;
background-color:#561CD4;
box-shadow: 10px 10px 5px grey;
}
Résultat

Border image

La propriété CSS border-image permet de définir une image comme bordure d'un élément. Elle offre une manière flexible et graphique d'appliquer des bordures, allant au-delà des styles de bordure traditionnels (solide, pointillé, etc.).

Exemple d'utilisation

Code

.borderimage {
    border: 20px solid transparent;
    padding: 15px;
    border-image: url(/assets/img/bg_borderimage.jpg) 30 stretch;
}

Columns count

La propriété column-count définit le nombre de colonnes dans lesquelles le contenu d'un élément doit être divisé. En utilisant cette propriété, le texte et les autres éléments enfants sont automatiquement répartis sur le nombre de colonnes spécifié.

Exemple d'utilisation

Code

.columns{
    column-count: 3;
    column-gap: 20px;
}

Filter

la propriété filter est utilisée pour appliquer des effets visuels à un élément HTML et à son contenu. Cette propriété permet de modifier l'apparence visuelle des éléments, tels que les images ou les arrière-plans, en utilisant divers filtres prédéfinis.

Exemple d'utilisation

Code

.filter1{
filter: blur(2px);
}
.filter2{
    filter: brightness(225%);
}
.filter3{
    filter: contrast(200%);
}
.filter4{
    filter: opacity(50%);
}

Keyframe

Permet de créer des animations complexes. Par exemple, l’animation suivante fait changer la couleur de fond d’un élément de orange à rose, puis à vert en 4 secondes en boucle.

Exemple d'utilisation

Code

@keyframes colorchange {
     0%   {background-color: rgb(255, 81, 0);}
     50%  {background-color: rgb(255, 0, 191);}
     100% {background-color: rgb(193, 231, 22);}
}
                            
.keyframe{
    animation: colorchange 4s infinite;
    height: 100px;
    width: 100px;
}

Selecteur first-child

Les sélecteurs avancés permettent de cibler des éléments spécifiques de manière plus précise..

Exemple d'utilisation

Code

.selecteur :first-child {
    color: blue;
    font-size: 1rem;
}
.selecteur :last-child {
    color: orange;
    font-size: 1.5rem;
    }

transform

La propriété transform en CSS permet de changer la façon dont les éléments sont affichés à l'écran. On peut les déplacer, les faire tourner, les agrandir ou les rétrécir, voire même les déformer pour donner l'illusion de perspective.

Code

.square {
background-color: #3498db;
}

.rectangle {
background-color: #e74c3c;
transform: scaleX(1.5);
}
.diamond {
background-color: #2ecc71;
transform: rotate(45deg);
}

.triangle {
width: 0;
height: 0;
border-left: 50px solid transparent;
border-right: 50px solid transparent;
border-bottom: 100px solid #9b59b6;
transform: rotate(180deg); 
}
.parallelogram {
background-color: #f1c40f;
transform: skew(20deg);
}
.hexagon {
width: 58px; 
height: 100px;
background-color: #e67e22;
position: relative;
margin: 50px;
}
.hexagon:before,
.hexagon:after {
content: "";
position: absolute;
width: 0;
border-left: 29px solid transparent;
border-right: 29px solid transparent;
}
.hexagon:before {
bottom: 100%;
border-bottom: 50px solid #e67e22;
}
.hexagon:after {
top: 100%;
width: 0;
border-top: 50px solid #e67e22;
}
Résultat
Carre Rectangle Diamond
Triangle Parallelogram Hexagon
Source
MDN - transform

Text shadow

Code

p {
    text-shadow: 4px 4px 2px #3498db;
}
